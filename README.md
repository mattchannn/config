## Prerequisites

## System Requirements

- git v2 or greater
- npm v6 or greater
- Node.js v16 or greater

All of the above requirements should be set in your OS Path. Run below commands to verify.

```bash
git --version
npm --version
node --version
```

## Run the development server

```npm
npm run dev
```
